## Project Summary

Analysis of Superstore sales data to understand sales performance, profit trends, customer segments, and future sales forecasting using Excel and Power BI.

---

## Overview

This project focuses on analyzing historical Superstore sales data and building an interactive Power BI dashboard to gain insights into sales trends, profitability, regional performance, and product categories. It also includes sales forecasting to support data-driven business decisions.

---

## Problem Statement

Retail businesses generate large volumes of sales data, but raw data alone does not clearly reveal trends or future performance. The goal of this project is to analyze past sales data and create visual insights and forecasts that help understand business growth, profitability, and demand patterns.

---

## Dataset

* Superstore sales dataset (CSV / Excel format)
* Includes:

  * Order and shipment details
  * Sales and profit values
  * Product category and sub-category
  * Customer segment
  * Region and state
  * Order date and ship date

* Removed duplicates and handled missing values
* Standardized column names and data formats

### Data Analysis & Visualization (Power BI)

* Built relationships between tables
* Created calculated measures using DAX
* Designed interactive visuals for sales and profit analysis
* Implemented time-series forecasting for future sales trends
